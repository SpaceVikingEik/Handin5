To run the program you firstly need to use the commandline go run server.go in the folder Handin5
This will start 3 servers up, which you can see in log.txt

After that run however many clients you want with go run client.go and follow the terminal commands to bid and get the results

After 20 secs a server will chrash, yet the clients will continue working

After ca 40 seconds, the auction will be over and the terminal responses will reflect this
